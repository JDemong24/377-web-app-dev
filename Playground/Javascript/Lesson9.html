JavaScript Lesson 9

The focus of this lesson is arrays. We will use a loop to print the contents of an array which contains a list of cars.

    Exercise 1

    Write the HTML and JavaScript code to allow the user to add a car to the existing array. Then call showCars() refresh the list of cars.

    Exercise 2

    Write a function to remove every other car in the array. Then call showCars() refresh the list of cars.
